## わかったこと
≪復習≫
- `git init`：リポジトリを新規に作る。
- `git diff <コミットA> <コミットB>`：コミットA→コミットBの差分を見る。
- `git diff`：ワークツリーとインデックス(ステージング領域)の差分を確認する。
- `git diff —cached`：インデックスとリポジトリ(HEAD)の差分を確認する。
- `git add -p <ファイル名>`：ハンク(未変更行の間に挟まれた部分的な変更部分)を選択的にaddする。
- `git reset —soft`：HEADに指定したコミットにリセットする。この時、インデックスとワークツリーは変更されない。
- `git reset —mixed`：インデックスをHEADの内容にリセットする。
- `git reset —hard`：インデックスとワークツリーをHEADの内容にリセットする。
